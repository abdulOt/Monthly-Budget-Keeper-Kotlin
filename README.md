# Monthly Budget Keeper 💰📊

A modern Android application designed to help users manage their personal finances with ease. Monthly Budget Keeper allows users to track income, expenses, budgets, and savings through a clean and intuitive interface, making it easier to stay in control of monthly spending.

> Developed by Abdul Hanan

---

## ✨ Features

### 💵 Income Management
- Add multiple income sources
- Edit or delete income entries
- Monthly income overview
- Income history

### 💸 Expense Tracking
- Record daily expenses
- Organize expenses by category
- Add notes to transactions
- Edit and delete expenses

### 📊 Budget Planning
- Set monthly budgets
- Track remaining budget
- Budget utilization progress
- Spending alerts (planned)

### 📈 Financial Insights
- Monthly spending summary
- Income vs. Expenses
- Savings calculation
- Visual charts and analytics

### 🏷️ Categories
- Food
- Shopping
- Transport
- Entertainment
- Bills
- Healthcare
- Education
- Custom categories

### 🔍 Transaction History
- Search transactions
- Filter by category
- Filter by date
- Monthly reports

### ☁️ Cloud Synchronization
- User authentication
- Secure cloud backup
- Multi-device support
- Automatic data synchronization

### 🎨 User Experience
- Material Design 3
- Responsive layouts
- Smooth animations
- Dark Mode support (planned)
- Easy-to-use interface

# Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/monthly-budget-keeper.git
```

2. Open the project in Android Studio.

3. Add your Firebase configuration:

```
google-services.json
```

4. Configure Firebase Authentication and Firestore.

5. Build and run the application.
